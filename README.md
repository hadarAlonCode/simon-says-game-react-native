
## Simon Says - React Native App
A simple "Simon Says" game

### Game screen:
![Game Screen](src/images/gameScreen.jpeg)


### features:

- The sequence from Simon will be totally random.
- The sequence from Simon will increase by 1 for every successful try.
- If the player fails the game will stop until the user press start again.
- the user can't press the buttons while Simon is doing his sequence.
- The game keeps the highest score.
